---
title: "How to Edit your Request Flow"
---

If you want to edit your request flow, click the **Request** button on the side panel. Select the specific request you want to update. Then, in the top right corner, click the menu to access the editing options.

You can:

- Rename the request
- Change the due date
- Change the champ
- Change gatherers
- Change approvers
- Cancel the request
